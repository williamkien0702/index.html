This project showcases a custom programming language inspired by moods and emotions.

Features

  • Friendly natural language-inspired syntax
  • Arithmetic operations using emotional terms (elevate, drain, etc.)
  • Support for variables and mood-based values
  • Looping constructs (for loop, mood-based while loop)
  • Conditional logic with mood comparisons
  • Custom moods defined by the user
  • Fun output with say statements
